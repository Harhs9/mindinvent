 MindInvent Technologies Website

A modern, responsive business landing page built using Bootstrap that showcases the services, features, pricing, and client testimonials of MindInvent Technologies.

 Project Overview

This project is a static website designed to represent a technology company. It includes:

Responsive navigation bar
Hero section with call-to-action
Features/services section
Pricing plans
Testimonials
Contact form
Footer with social links

The design focuses on clean UI, responsiveness, and user-friendly layout.
